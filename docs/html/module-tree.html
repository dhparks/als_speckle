Module Hierarchy

  • speckle: A speckle correnation library for Beamline 12.0.2 at the Advanced Light Source at Lawrance Berkeley National Laboratory.
    • speckle.averaging: A Library for image averaging of various functions.
    • speckle.conditioning
    • speckle.crosscorr: Functions that calculate the image cross-correlation between two images.
    • speckle.fit: A general-purpose library for fitting image and (x,y) data.
    • speckle.gpu
    • speckle.gpu_domain_generator
    • speckle.io: Library for input and output for coherent measurements at beamline 12.0.2 at the Advanced Light Source.
    • speckle.kernels
    • speckle.phasing
    • speckle.propagate
    • speckle.scattering: A Library for calculating various scattering parameters such as Flangosaurus distances, Q, and energy to wavelength conversion.
    • speckle.shape: A library for calculating image shapes and functions
    • speckle.wrapping: A Library for wrapping and unwrapping arrays that have circular symmetry.
    • speckle.xpcs: Library for XPCS and photon correlation experiments.
